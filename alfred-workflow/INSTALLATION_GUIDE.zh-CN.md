# JSON Handle Alfred 工作流 - 安装指南

## 概述

本工作流实现 HTTP 环境下的完全自动化 JSON 处理流程，通过权限检测、会话识别、自动粘贴等实现无缝体验。

## 功能特点

- **完全自动化**：HTTP 环境下一键完成
- **权限管理**：自动检测并引导用户配置
- **会话识别**：唯一 ID 精确定位目标窗口
- **精确粘贴**：智能等待页面加载完成
- **无大小限制**：支持 MB 级 JSON
- **安全可靠**：数据不经过网络传输

## 安装步骤

### 步骤1：安装 Alfred 工作流

1. 找到 `JSON Handle.alfredworkflow` 文件
2. 双击 `.alfredworkflow` 文件
3. 在 Alfred 弹窗中点击 **"Import"** 确认导入
4. 验证：`Cmd + Space` 输入 `json` 应出现 "JSON Handle"

### 步骤2：配置辅助功能权限（必需）

**为什么需要**：自动粘贴需要模拟键盘操作，Alfred 需要辅助功能权限。

**配置方法**：
1. 系统偏好设置 → 安全性与隐私 → 隐私
2. 选择「辅助功能」
3. 点击锁图标解锁，点击 "+" 添加 Alfred
4. 确保 Alfred 勾选已启用

### 步骤3：测试工作流

```bash
cd alfred-workflow
./test_workflow.sh
```

## 使用方法

1. **复制 JSON**：将 JSON 复制到剪贴板
2. **触发**：`Cmd + Space` → 输入 `json` → 回车
3. **自动完成**：浏览器打开 → 页面加载 → 自动粘贴 → JSON 解析

**总耗时约 1-2 秒**

## 故障排除

- **权限错误**：按步骤2配置辅助功能权限
- **自动粘贴失败**：增加延迟时间或检查默认浏览器
- **会话超时**：确保页面完全加载
- **大 JSON 失败**：增加粘贴延迟或压缩 JSON
